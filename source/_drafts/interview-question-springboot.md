## Q1 springboot有哪些特色
### 回答思路
1. Spring boot 产生的契机
2. Spring boot 是什么
3. Spring boot 特点和原理

### Spring boot 产生的契机
> Spring boot 是在spring4.0的基础上产生的，意在简化开发模式是开发者快速开发出基于Spring 的应用。SpringBoot的出现使得开发人员不再拘于繁琐的模板化的配置，可以快速搭建应用。
### Spring boot 的特点
- 自动配置Spring
- 简化Maven配置，整合常用依赖
- 可自动嵌入中间件插件
- 提供应用状态
